HDFView 2.14.0 with hdf-java version 3.3.2
------------------------------------------------------------------------------

This directory contains the binary (release) distribution of
HDFView 2.14 that was compiled on:
       Windows win-amd64

with:  Java JDK 1.8.0_66

It was built with the following options:
    -- STATIC HDF 4.2.13
    -- STATIC HDF5 1.8.19

===========================================================================
Note: By default HDFView runs on the included Java JRE 1.8.
      This can be modified by changing JAVABIN in HDFView\2.14.0\hdfview.bat
===========================================================================

The contents of this directory are:

    COPYING                 - Copyright notice
    README.txt              - This file
    HDFView-2.14.msi       - HDFView Installer

Running
===========================================================================
To install HDFView for Windows:

1. Execute Execute HDFView-2.14.msi
2. Follow prompts
3. Execute HDFView\2.14.0\hdfview.bat to run HDFView
===========================================================================
